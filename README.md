## Quick Start

First of all, give it a star, pleaseðŸ˜. Then, fork this repo.

### Usging npm/yarn/pnpm

1. Install dependencies:

```sh
npm i

# or

yarn

# or

pnpm i --shamefully-hoist
```

2. Running in dev

```sh
npm run dev

# or

yarn dev

# or

pnpm run dev
```

3. Build the project

```sh
npm run build

# or

yarn build

# or

pnpm run build
```

> The output files will be in `.output` folder.

4. Preview

```sh
npm run preview

# or 

yarn preview

# or

pnpm run preview

# or

node .output/server/index.mjs
```
